# EffortlessHire
Goal 1: uni-appで募集系のWechatのAPPを作成する。 

Goal 2: 運営者は企業から人員探しができたら、費用をもらえる。そのお金で応募成功者へ一部奨励金として返礼する。 

Goal 3: 管理者１人でも普段の仕事が終わった時間で運営できる。
